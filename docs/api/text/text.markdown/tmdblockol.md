A markdown block ordered list


## Methods

### `Method Start:UInt()`

Start index of the ordered list.

<br/>

### `Method IsTight:Int()`

Non-zero if tight list, zero if loose.

<br/>

### `Method MarkDelimiter:Int()`

Character delimiting the item marks in MarkDown source.

e.g. `.` or `)`


<br/>
