A markdown table block.


## Methods

### `Method ColCount:UInt()`

The number of columns in the table.

<br/>

### `Method HeadRowCount:UInt()`

The number of header rows.

Always returns 1.


<br/>

### `Method BodyRowCount:UInt()`

The number of body rows.

<br/>
