Represents a sheet that has been packed with rectangles.


## Fields

### `Field width:Int`

The width of the sheet.

<br/>

### `Field height:Int`

The height of the sheet.

<br/>

### `Field rects:SPackedRect[]`

The rectangles that have been packed into the sheet.

<br/>
