Represents a rectangle that has been packed into a sheet.


## Fields

### `Field id:Int`

The ID of the rectangle.

<br/>

### `Field x:Int`

The X position of the rectangle.

<br/>

### `Field y:Int`

The Y position of the rectangle.

<br/>

### `Field width:Int`

The width of the rectangle.

<br/>

### `Field height:Int`

The height of the rectangle.

<br/>

### `Field rotated:Int`

Whether the rectangle has been rotated.

<br/>
