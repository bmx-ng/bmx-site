Int holder for key returned by TIntMap.Keys() enumerator.


Because a single instance of [TIntKey](../../../collections/collections.intmap/tintkey) is used during enumeration, [value](../../../collections/collections.intmap/tintkey/#field-valueint) changes on each iteration.


## Fields

### `Field value:Int`

Int key value.
