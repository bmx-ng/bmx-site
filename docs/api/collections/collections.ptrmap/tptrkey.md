Int holder for key returned by TPtrMap.Keys() enumerator.


Because a single instance of [TPtrKey](../../../collections/collections.ptrmap/tptrkey) is used during enumeration, [value](../../../collections/collections.ptrmap/tptrkey/#field-valuebyte-ptr) changes on each iteration.
